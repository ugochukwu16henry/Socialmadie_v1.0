# Socialmadie v1.0 Workspace

This workspace has been organized to keep external platform repositories together for later reference and integration.

## Integrated Repositories

All cloned repositories now live under `integrations/`:

- `integrations/n8n` — automation/workflow engine source
- `integrations/mixpost` — social publishing platform source
- `integrations/randolly` — additional source extracted for review/use
- `integrations/activepieces` — automation platform source

## Current Structure

```text
Socialmadie_v1.0/
├── integrations/
│   ├── activepieces/
│   ├── mixpost/
│   ├── n8n/
│   └── randolly/
└── README.md
```

## Note

On Windows, the `activepieces` repository may show two uncheckoutable screenshot files with `:Zone.Identifier` in their names. The rest of the repository is available and usable.
